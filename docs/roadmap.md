# Roadmap

## Phase 1

- Telegram bot shell
- Pasted text parser
- Minimum deterministic scoring engine
- Stable curatorial card

## Phase 2

- Larger local knowledge base
- Persistent cache
- More regression playlists
- More precise scene relationships

## Phase 3

- REST API hardening
- Web dashboard
- Export to Markdown, PDF, and image cards
- Structured observability

## Phase 4

- External enrichment providers
- Feedback loop
- Contextual comparisons
- Human-reviewed data curation tools
